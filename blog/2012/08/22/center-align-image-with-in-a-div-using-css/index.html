Center Align Image With in a Div Using Css

Aug 22nd, 2012

HTML Code

<div class="container">
  <img src="#" />
</div>

CSS Code

.container {
  position: relative;
}
img {
  position: absolute;
  top: 0; left: 0; right: 0; bottom: 0;
  margin: auto;
}
